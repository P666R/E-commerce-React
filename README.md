# Installation And Usage:

## To run this project run

- git clone https://github.com/P666R/E-commerce-React.git

## Go to directory

- cd E-commerce-React-main

## Install all dependencies

- npm install

## Run Project

- npm run dev

## Now you can access on localhost:5173
